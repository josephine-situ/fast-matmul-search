# fast-matmul-search

Repository scaffold for exploring fast matrix multiplication search methods.

## Structure

- `src/`: core implementation modules
- `scripts/`: runnable helpers and validation entry points
- `results/`: generated outputs and summaries
- `notebooks/`: interactive exploration notebooks
- `tests/`: unit and integration tests

## Next steps

Implement the search, validation, and analysis modules, then use the scripts and tests to reproduce and compare candidate algorithms.
